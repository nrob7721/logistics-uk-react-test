# Desciption

Simple React app that displays menu items and driver data dynamically from JSON files. 

# Implemented and Learned

- Bootstrapping a React app with Create React App
- Integrated JSON data sources and performed aggregation of driver working, driving, and idle times
- Strongly typed interfaces via use of typescript
- Implemented reuseable react components
- Client-side routing using React Router
- Practiced building a small maintainable front-end project from scratch

# Scripts

- `npm run start` - Runs the app in the development mode.
